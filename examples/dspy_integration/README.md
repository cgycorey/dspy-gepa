# DSPY Integration Examples

This directory contains examples of using DSPY-GEPA integration for optimizing DSPY programs.

## Files

- `dspy_example.py` - Comprehensive DSPY integration example
- `simple_dspy.py` - Simplified DSPY example (coming soon)

## Requirements

```bash
# Install DSPY (optional)
uv add dspy
```

## Quick Start

```python
# See the full example
uv run python examples/dspy_integration/dspy_example.py
```

## What's Demonstrated

1. Creating DSPY programs
2. Converting to GEPA candidates
3. Multi-objective optimization
4. Evolutionary improvement of DSPY programs

## Note

DSPY is an optional dependency. The examples include mock DSPY classes for demonstration when DSPY is not installed.
